# SoulGrave

日本語 | [한국어](#한국어)

プレイヤーが死亡した際に、その場へ「墓」を生成する Paper / Minecraft プラグインです。
墓にはデスポイントで失ったアイテムと経験値が保管され、所有者が回収できます。

## 特徴

- 死亡地点付近の安全な場所に墓ブロック（ソウルサンド・クォーツ・ウィザースケルトンの頭）を設置
- 墓の上にホログラム（所有者名・生成日時・消滅予定）を表示
- 安全な場所が見つからない場合は通常ワールドにランダム生成（緊急墓）
- 通常の墓は 24 時間、緊急墓は 1 時間で自動消滅（時間は固定）
- 墓ごとに `.yml` ファイルでデータを永続化し、サーバー再起動後も復元
- `/grave list [player]` で墓を管理（権限: `soulgrave.admin`）

## 動作環境

- サーバー: Paper `26.1.2`（Paper 系サーバー）
- Java: 25 以上（実行環境）
- 設定ファイル: なし（設定不要、導入するだけで動作します）

## 使い方（動作概要）

1. プレイヤーが死亡すると、死亡地点付近の安全な場所に墓が自動生成されます。
2. 墓の上には、所有者名・生成日時・消滅予定時刻がホログラムで表示されます。
3. 周囲に安全な場所が見つからない場合は、通常ワールドにランダムで「緊急墓」が生成されます。
4. 墓を破壊するか右クリックすると、保管されていたアイテムと経験値が戻ります（詳細は「回収方法」を参照）。
5. 通常の墓は 24 時間、緊急墓は 1 時間で自動的に消滅します。
6. 墓のデータは `.yml` ファイルに保存されるため、サーバーを再起動しても消えません。

## 回収方法

墓を構成するブロック（ソウルサンド・クォーツ・ウィザースケルトンの頭）のいずれか 1 つを、
破壊するか右クリックすることで回収できます。回収すると、保管されていたアイテムと経験値が戻ります。

- 右クリック: 所有者本人、または OP のみ回収できます。
- ブロック破壊: 所有者以外のプレイヤーでも回収できます。他のプレイヤーが破壊して回収した場合、
  オンラインの所有者に通知が送られます。

## インストール

1. JAR ファイルを Paper サーバーの `plugins/` フォルダに配置します。
2. サーバーを再起動します。

> 現在、配布用のビルド済み JAR は準備中です。ビルド方法が分からない場合は、お気軽に [Issues](../../issues) からお問い合わせください。

## ビルド（開発者向け）

[Maven](https://maven.apache.org/) と JDK 25 が必要です。

```
mvn clean package
```

ビルド成果物 `target/SoulGrave-1.1.0.jar` が生成されます。

| コマンド | 説明 | 権限 |
| --- | --- | --- |
| `/grave list [player]` | 墓の一覧表示 | `soulgrave.admin` (default: op) |

# 한국어

[日本語](#soulgrave) | 한국어

플레이어가 사망했을 때 그 자리에 "무덤"을 생성하는 Paper / Minecraft 플러그인입니다.
무덤에는 사망 지점에서 잃은 아이템과 경험치가 보관되며, 소유자가 회수할 수 있습니다.

## 특징

- 사망 지점 근처의 안전한 장소에 무덤 블록(소울 샌드·석영·위더 스켈레톤 머리)을 설치
- 무덤 위에 홀로그램(소유자 이름·생성 일시·소멸 예정)을 표시
- 안전한 장소를 찾지 못한 경우 일반 월드에 무작위 생성(긴급 무덤)
- 일반 무덤은 24시간, 긴급 무덤은 1시간 후 자동 소멸(시간 고정)
- 무덤별로 `.yml` 파일에 데이터를 영속화하여 서버 재시작 후에도 복원
- `/grave list [player]` 로 무덤 관리(권한: `soulgrave.admin`)

## 동작 환경

- 서버: Paper `26.1.2`(Paper 계열 서버)
- Java: 25 이상(실행 환경)
- 설정 파일: 없음(설정 불필요, 도입만 하면 동작합니다)

## 사용 방법(동작 개요)

1. 플레이어가 사망하면 사망 지점 근처의 안전한 장소에 무덤이 자동 생성됩니다.
2. 무덤 위에는 소유자 이름·생성 일시·소멸 예정 시각이 홀로그램으로 표시됩니다.
3. 주변에 안전한 장소가 없을 경우, 일반 월드에 무작위로 "긴급 무덤"이 생성됩니다.
4. 무덤을 파괴하거나 우클릭하면 보관되어 있던 아이템과 경험치가 돌아옵니다(자세한 내용은 "회수 방법" 참조).
5. 일반 무덤은 24시간, 긴급 무덤은 1시간 후 자동으로 소멸합니다.
6. 무덤 데이터는 `.yml` 파일에 저장되므로 서버를 재시작해도 사라지지 않습니다.

## 회수 방법

무덤을 구성하는 블록(소울 샌드·석영·위더 스켈레톤 머리) 중 하나를
파괴하거나 우클릭하면 회수할 수 있습니다. 회수하면 보관되어 있던 아이템과 경험치가 돌아옵니다.

- 우클릭: 소유자 본인 또는 OP만 회수할 수 있습니다.
- 블록 파괴: 소유자가 아닌 플레이어도 회수할 수 있습니다. 다른 플레이어가 파괴하여 회수한 경우,
  온라인 상태인 소유자에게 알림이 전송됩니다.
