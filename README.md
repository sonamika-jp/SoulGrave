# SoulGrave

プレイヤーが死亡した際に、その場へ「墓」を生成する Paper / Minecraft プラグインです。
墓にはデスポイントで失ったアイテムと経験値が保管され、所有者が回収できます。

## 特徴

- 死亡地点付近の安全な場所に墓ブロック（ソウルサンド・クォーツ・ウィザースケルトンの頭）を設置
- 墓の上にホログラム（所有者名・生成日時・消滅予定）を表示
- 安全な場所が見つからない場合は通常ワールドにランダム生成（緊急墓）
- 通常の墓は 24 時間、緊急墓は 1 時間で自動消滅
- 墓ごとに `.yml` ファイルでデータを永続化し、サーバー再起動後も復元
- `/grave list [player]` で墓を管理（権限: `soulgrave.admin`）

## ビルド

[Maven](https://maven.apache.org/) と JDK 25 が必要です。

```bash
mvn clean package
```

ビルド成果物 `target/SoulGrave-1.1.0.jar` を生成します。

## インストール

1. ビルドした JAR を Paper サーバーの `plugins/` フォルダに配置
2. サーバーを再起動

対応 API バージョン: Paper `26.1.2`

## コマンド・権限

| コマンド | 説明 | 権限 |
|----------|------|------|
| `/grave list [player]` | 墓の一覧表示 | `soulgrave.admin` (default: op) |
